# COMP 5700 – AI Pull Request Security Analysis Project

This repository contains the code and results for Tasks 1–5 of the COMP 5700 course project.  
The objective of this project is to analyze a dataset of pull requests generated by agentic AI tools and determine which tasks are security-related.

## Repository Structure

│ ├── all_pull_request.csv

├── src/ # Source code for each task

│ ├── task4.py

## Tasks Included

### Task 1
Extract pull request metadata from `all_pull_request.csv`.

### Task 2
Extract repository metadata from `all_repository.csv`.

### Task 3
Extract task-type metadata from `pr_task_type.csv`.

### Task 4
Extract commit details from `pr_commit_details.csv`.

### Task 5
Combine processed data from Tasks 1–4 into a unified CSV with:
- ID  
- AGENT  
- TYPE  
- CONFIDENCE  
- SECURITY (boolean flag)

## Dataset Source

HuggingFace dataset link:  
https://huggingface.co/datasets/hao-li/AIDev

python3 src/task2.py
